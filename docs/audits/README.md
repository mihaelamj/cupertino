# `docs/audits/`

This directory holds dated audit + release-readiness artifacts that
accumulate over time. Anything generated by a session-bound audit pass
(closure replay, open-issue probe, regression-lock inventory, release
readiness, doc audit) lives here.

## Convention

- **Filename shape**: `<kind>-<scope>-<YYYY-MM-DD>.md` for dated audits,
  `<kind>-<scope>-<version>.md` for release-bound artifacts.
  Lowercase + hyphenated. Examples:
  - `closure-replay-2026-05-17.md`
  - `open-issues-probe-2026-05-17.md`
  - `stage-d-regression-locks-2026-05-17.md`
  - `release-readiness-v1.2.0.md`
- **NOT at repo root.** Pre-#733 the first batch of audits landed at the
  repo root (`ISSUE-AUDIT-*.md`, `DOC-AUDIT-*.md`, `RELEASE-READINESS-*.md`).
  Those got moved here in PR for #733 and renamed lowercase-hyphen for
  consistency. Future audits go here directly.
- **Not auto-pruned.** These docs are session-history artifacts — they
  describe what was true on a specific date, not the current state.
  Older audits stay around as a paper trail; the most recent one is the
  authoritative current snapshot.

## Relation to the `cupertino` audit methodology

Each Stage of the ironclad-sweep methodology produces one doc here:

- **Stage A** — closure replay (re-verify recent closures against acceptance)
- **Stage B** — older-closures sample (deferred, currently not run)
- **Stage C** — open-issue currency probe
- **Stage D** — regression locks (test/lint/CI guards per confirmed closure)
- **Stage E** — process changes (closure protocol template) — deferred

The methodology itself is described in `closure-replay-2026-05-17.md`'s
`## Method` section. Future audit passes should follow the same shape.

## Adding a new audit

1. Pick the canonical filename per the convention above.
2. Drop the markdown in this directory (NOT at repo root).
3. Cross-link from CHANGELOG if the audit produced shipped changes, or
   from the PR body that the audit triggered.
4. Don't backfill old audits into the new naming — leave the historical
   paper trail in place.
