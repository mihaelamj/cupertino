# Packages View

Browse and search Swift packages.

## Description

Browse the Swift packages corpus in `packages.db` (183 Apple-official packages with full source files + GitHub metadata; the previous embedded 9,699-URL catalog was removed in #194 — `packages.db` is now the canonical source, shipped via `cupertino setup`). Search, view details, and mark packages for download.

## Features

- Full-text search across package names and descriptions
- View package metadata (stars, license, description)
- Mark packages for download
- Filter and sort options

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `↑` / `k` | Move selection up |
| `↓` / `j` | Move selection down |
| `/` | Start search |
| `Enter` | View package details |
| `Space` | Toggle selection |
| `h` / `Esc` | Return to home |
| `q` | Quit |

## Search

Press `/` to enter search mode. Type your query and press `Enter` to search. Press `Esc` to cancel.
