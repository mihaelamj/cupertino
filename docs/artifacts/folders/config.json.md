# config.json

Application configuration file

## Location

```
~/.cupertino/config.json
```

## Purpose

Stores user configuration and preferences for Cupertino.

## Format

```json
{
  "version": "1.0",
  "settings": {
    // User preferences
  }
}
```

## Created By

- Generated automatically on first run (if needed)
- Can be manually created/edited

## Usage

Currently defined but not actively used. Reserved for future configuration features such as:
- Default crawl settings
- Search preferences
- Output format preferences
- API keys/tokens

## Related

- Application constants defined in `Sources/Shared/Constants.swift`
